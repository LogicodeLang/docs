Logicode

Welcome to the Logicode Documentation.

Logicode is a minimalistic language, based on three logic gates:

  • AND (&)
  • OR (|)
  • NOT (!)

The rest, you have to do yourself.

You can make all sorts of things in Logicode, like circuits and conditionals.

Logicode provides you with tools, while you build whatever you need.

Sample Code (w/ commentary):


      circ asdf(n) -> [  # Defines a circuit called asdf, with 1 argument (n)
          var jk = 1011  # Creates a variable called jk, set to 1011
          jk&n           # Calculates AND of arg "n" and jk
      ]                  # End circuit
      out asdf(100)      # Outputs asdf(100)
